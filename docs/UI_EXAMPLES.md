# UI Examples Summary

These examples exist as references for layout and styling.

## Included reference screens

### 1. Landing page reference
A premium landing page for an RSS-to-social AI product with:
- hero section
- value proposition
- three-step workflow
- visual example of article to post transformation
- pricing section
- CTA section

### 2. Feed configuration reference
A setup page with:
- feed name
- RSS URL
- verification action
- include/exclude keywords
- minimum word count
- refresh rate
- AI personalization settings
- platform-specific generation settings

### 3. Post editor reference
A review screen with:
- source content panel
- generated post editor
- platform tabs
- regenerate action
- alternate hooks
- schedule and publish actions

### 4. Connected accounts reference
An accounts page with:
- connected account cards
- connection status
- settings and disconnect actions
- add connection CTA

## Build expectation
Use these references to create polished REZZUM pages in the app.
Do not preserve the old product name from the examples.
Do not hardcode fake content except as development seed/demo data.
